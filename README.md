# 🛡️ PhishGuard – Real-Time Phishing Detection System



## 🧠 Overview
PhishGuard is a cybersecurity-focused system designed to detect and block phishing attacks in real time. It integrates a Chrome extension with a Python-powered backend and a React dashboard to proactively protect users from malicious websites.

Unlike traditional systems, PhishGuard blocks suspicious pages **before they fully load**, reducing exposure to phishing threats.

---

## 🔥 Key Features

### 🧩 Chrome Extension Protection
- Detects phishing URLs in real time  
- Blocks malicious pages before full load  
- Provides instant user alerts  

### 🧠 Intelligent Threat Detection
- Python-based backend for URL analysis  
- Identifies suspicious patterns and domains  
- Designed for real-world phishing scenarios  

### 📊 Dashboard & Monitoring (React)
- Displays logs of scanned URLs  
- Tracks flagged and safe links  
- Provides user-friendly interface for monitoring  

### 🔐 Quarantine System
- Quarantine button for suspicious links  
- Allows users to isolate and review threats  
- Maintains history of flagged URLs  

### 🎓 User Training Module
- Built-in MCQ-based learning system  
- Educates users about phishing techniques  
- Improves awareness and decision-making  

---

## 🏗️ System Architecture

User → Chrome Extension → Python Backend → Threat Analysis  
→ Block / Allow Decision → React Dashboard (Logs + Monitoring)

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Python  
- **Extension:** Chrome Extension APIs
